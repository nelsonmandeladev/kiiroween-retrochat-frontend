# Notification Sounds

This directory contains notification sound files for the application.

## Required Sound Files

- `message.mp3` - Played when receiving new messages (1-on-1 and group)
- `friend-request.mp3` - Played when receiving friend requests
- `group-invite.mp3` - Played when invited to a group

## Adding Custom Sounds

Replace the placeholder files with your own sound files. Recommended format is MP3 with a duration of 0.5-2 seconds.

You can find free notification sounds at:

- https://notificationsounds.com/
- https://freesound.org/
- https://mixkit.co/free-sound-effects/notification/
